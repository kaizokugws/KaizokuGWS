## About

The Amazing Spider-Man 2 is an open-world action-adventure game based on the 2014 film. Set after the events of the movie, the game features an original storyline with Spider-Man facing off against iconic villains including Electro, Rhino, and the Green Goblin. The game features an expanded open-world New York City to explore with improved web-slinging mechanics, new combat abilities, and a branching storyline that lets players choose how to handle various threats across the city.
